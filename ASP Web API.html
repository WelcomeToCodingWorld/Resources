ASP.NET Web API is a framework that makes it easy to build HTTP services that reach a broad range of clients, including browsers and mobile devices. ASP.NET Web API is an ideal platform for building RESTful applications on the .NET Framework. (ASP.NET Web API是一种框架，用于轻松构建基于多种客户端（包括浏览器和移动设备）的Http服务,是一种在.NET Framework上构建RESTful应用程序的理想平台)
